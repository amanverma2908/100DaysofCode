# Floyd's Triangle

Today, I learnt about the concept of Floyds Triangle.

## Task
Print the floyds triangle (pattern)

## Example
```bash
     1 
    1 1
   1 2 1 
  
 ```
  

### Input Format
  
  A single integer, n for the number of rows.

### Constraints
  * 0 < n < 10 <sup>3</sup>

_ _ _ _

Input

  Enter the number of rows: 5
